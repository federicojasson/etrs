----------------------------

- Warnings.
- Multiple spaces instead of tabulations.
- Double semicolons.
- File includes.

- if (!...) instead of if (! ...) (regular expression: ![^ ]).
- /* instead of /**.
- Multiple blank lines (regular expression: [\t]*\n[\t]*\n[\t]*\n).
