Release tasks
----------------------------

- Check PHP and MySQL versions in which the system was tested.
- Configure the cron-job table.
- Change the password of database users.
- Check if APC cache is installed.
- Configure httpd.conf:
	ServerTokens ProductOnly
	ServerSignature Off
- Configure php.ini:
	upload_max_filesize=0
	post_max_size=0
	expose_php=Off
- Configure my.ini (DBMS cache):
	query_cache_limit = 64K
	query_cache_size = 20MB
	query_cache_type = 1
- Register domain.
- Buy SSL/TSL certificate.
